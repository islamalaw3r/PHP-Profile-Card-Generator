# PHP Profile Card Generator

## Project Description

This project is a simple Profile Card Generator developed using PHP.

The application demonstrates the use of:

* PHP Arrays
* PHP Functions
* PHP Loops
* HTML5
* CSS3

The profile cards are generated dynamically from PHP data structures.

---

## Technologies Used

* PHP
* HTML5
* CSS3

---

## Features

* Generate multiple profile cards
* Display profile image
* Display name
* Display job title
* Display email
* Display additional profile information
* Clean and responsive layout

## How to Run

1. Install XAMPP.
2. Start Apache.
3. Copy the project folder into:

```
C:\xampp\htdocs\
```

4. Open your browser and visit:

```
http://localhost/PHP-Profile-Card-Generator/
```
